# Autorent

Seadistasin autorendi projekti Ubuntu Serveri peal.

Kasutasin:
- Nginx
- PHP
- MariaDB
- Git

Tegin järgmised tegevused:
- kloonisin projekti GitHubist
- paigaldasin vajalikud paketid
- lõin andmebaasi `car_rent`
- importisin SQL faili
- seadistasin `config.php` faili õigete andmebaasi andmetega
- panin veebilehe tööle brauseris
- tegin andmebaasist SQL dump faili

Veebileht avaneb Ubuntu virtuaalmasina IP aadressi kaudu brauseris.

Projekti juures on olemas:
- muudetud kood
- SQL dump fail
- andmebaasi ühendus
